Appcelerator

Javascript development platform for creating applications for iOS, Android, Windows Phone and Blackberry

Features:

  • Robust IDE with WYSIWYG UI editor, can build native apps for every mobile OS
  • Included Arrow UI allows you to quickly build Databases and models for the application
  • Easy to use mobile test automation
  • Included application dashboard to track user metrics and other analytics
  • Easy to use push notification service to quickly notify users

Pricing:

Free trial available
Single Developer: $39/Month
Teams: $250/Developer/Month

Titanium IDE

  • MVC framework built in
  • Virtual Phone Simulators let you see what your app looks like as you build it
  • Automated tests are built in to the IDE and test cases can easily be written

Arrow Backend

  • Can connect to SQL/MongoDB/Azure and also has cloud for you to upload to
  • You can easily build your own API or use existing API's
  • GUI allows you to see what your model looks like while you build it

Analytics

  • Automatic Data Collection and Aggregation
  • You can create your own dashboards to measure and track whatever data you want
  • Prebuilt dashboards allow you to easily see usage statistics and user uptime

Pros:

  • Full featured enterprise ready application development solution
  • Attractive easy to use UI
  • Easy to set up and maintain Backend and API's with Arrow

Cons:

  • May be paying for unneeded features
  • No choice of toolkits
  • Very expensive for development teams
