Appcelerator

Javascript development platform for creating applications for iOS, Android, Windows Phone and Blackberry

Features:

  • Robust IDE with WYSIWYG UI editor, can build native apps for every mobile OS
  • Included Arrow UI allows you to quickly build Databases and models for the application
  • Easy to use mobile test automation
  • Included application dashboard to track user metrics and other analytics
  • Easy to use push notification service to quickly notify users
